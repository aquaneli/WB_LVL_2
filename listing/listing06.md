Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передаче их в качестве аргументов функции.

```go
package main
 
import (
  "fmt"
)
 
func main() {
  var s = []string{"1", "2", "3"}
  modifySlice(s)
  fmt.Println(s)
}
 
func modifySlice(i []string) {
  i[0] = "3"
  i = append(i, "4")
  i[1] = "5"
  i = append(i, "6")
}

```

Ответ:
```
Программа выведет [3 2 3]. Мы создали слайс строк []string{"1", "2", "3"} , его len и cap равны 3. При вызове функции modifySlice(i []string) в качестве аргумента передали этот слайс. Внутри функции создалась копия слайса, но указатель внутри созданного слайса i []string указывает на те же элементы что и родительский слайс s []string. Когда присвоили значение элемента с индексом [0] то в обоих слайса поменяли значение. Затем к i []string добавили элемент, так как мы вышли за cap то указатель внутри i []string указывает на другой слайс и имеет cap 6. Теперь s []string и i []string имеют два разных указателя и любые изменения в i []string не изменят значения элементов s []string.
Слайс имеет len , cap и указатель на первый элемент массива строк. При передаче их в качестве аргумента копируется все содержимое слайса, указатель внутри слайса указывает на те же элементы что и слайс переданный в качестве аргумента.
```
